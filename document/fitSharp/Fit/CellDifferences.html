Cell Differences
When a value from the system under test does not match an expected value in a cell, the actual value is displayed and the cell is marked as an error.

when test is
sample domain

setinteger field2
checkinteger field3

setstringssome,thing
checkstringsor,other
then result is
sample domain

setinteger field2
checkinteger field3 expected
2 actual

setstringssome,thing
checkstringsor,other expected
some, thing actual

If the values being compared are strings, a message describing the difference is displayed.

when test is
sample domain

setstring fieldBob
checkstring fieldBub
then result is
sample domain

setstring fieldBob
checkstring fieldBub expected
Bob actual
At 1 expected u was o
